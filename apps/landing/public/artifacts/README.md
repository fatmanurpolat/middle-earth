# Landing artwork

Drop image files here with these exact names. The landing page renders a
graceful elven-frame fallback for any file that is missing, so the site works
even before the images are added.

| File                          | Where it appears        | Subject                                              |
| ----------------------------- | ----------------------- | --------------------------------------------------- |
| `middle-earth-haven.png`      | Hero "artifact" frame   | Wide haven panorama (the original screenshot)        |
| `rivendell-valley.png`        | Gallery — Rivendell     | Green Rivendell valley with waterfalls & elven halls |
| `the-one-ring-forging.png`    | Gallery — The One Ring  | The Ring heating up (triptych, portrait)             |
| `rivendell-autumn.png`        | Gallery — Imladris      | Rivendell in autumn / sunset light                   |
| `great-eagle-orthanc.png`     | Gallery — The Eagles    | An Eagle (Gwaihir) over the tower of Orthanc         |
| `the-one-ring-grass.png`      | Gallery — Ring in green | The One Ring in grass, Elvish inscription aglow      |

PNG or JPG both work — keep the filename (including the `.png` extension) as
listed above, or update `src/assets/artifact.ts` if you prefer different names.
